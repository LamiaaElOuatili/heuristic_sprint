# Heuristic MBVST Project

This repository contains the code for solving the Minimum Branching Vertex Spanning Tree (MBVST) problem using a lazy PLE approach and heuristic reconnection for large graphs.

## Test Instances

All test instances are stored in the folder: Spd_RF2_20_27_211

You can specify which test file to use directly in `main.py`.

## How to Run

1. Open `main.py`.  
2. Specify the name of the test file you want to use by editing the corresponding variable in `main.py`. For example:

```python
test_file = "Spd_RF2_20_27_211/test_instance_01.txt"
```
3. Run the script:

python main.py
